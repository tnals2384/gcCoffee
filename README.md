### 로컬 카페 Grids & Circles
Coffe Bean package를 온라인 웹 사이트로 주문합니다. 매일 전날 오후 2시부터 오늘 오후 2시까지의 주문을 모아서 처리합니다.

### 기술
Spring Boot, Mybatis, MySQL
Mybatis 사용 방식을 익히기 위한 프로젝트이므로, Mybatis 사용에 목적을 두었습니다.
로그인 기능 X
